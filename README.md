Business Sales Analysis Dashboard (Power BI)
🚀 Project Overview

This project presents an interactive Power BI dashboard built to analyze regional sales performance, customer behavior, and revenue trends.

It transforms raw sales data into meaningful insights using SQL queries and visual storytelling.

🧠 Key Insights
📈 Monthly sales trends and growth patterns
🌍 Region-wise loss percentage (Cancelled & Returned orders)
💸 Revenue loss by product and region
🛒 Average order value by category
🧑‍💼 Top-performing sales agents
🧮 Category-wise contribution to total sales
🔁 Customers with high return frequency
🗄️ Data Source

The dataset (RegionalSales2025) includes:

Order Date
Region
Product Name
Category
Sales Agent
Order Status
Total Amount
Customer ID
🧾 SQL Analysis

Key queries used for insights:

Monthly Sales Aggregation
Regional Loss Percentage
Top Revenue Loss Products
Average Order Value
Top Sales Agents
Category Contribution %
Frequent Return Customers

(Full SQL scripts available in project files)
